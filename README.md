# Plateforme Ibn Sina 🎓

Plateforme de gestion communautaire pour les étudiants, délégués et lauréats du lycée Ibn Sina. Cette application web permet de gérer les groupes WhatsApp, les événements, le mentorat et l'annuaire des anciens élèves.

## 🚀 Fonctionnalités

- **Authentification Sécurisée** : Connexion par email/mot de passe avec gestion des rôles (Firebase Auth).
- **Tableaux de Bord par Rôle** :
  - 👨‍🎓 **Étudiants** : Accès aux groupes WhatsApp, événements, demandes de mentorat.
  - 🎓 **Lauréats (Alumni)** : Réseau, partage d'expérience, événements dédiés.
  - 📋 **Délégués** : Gestion des étudiants de leur filière.
  - 🛠️ **Admin** : Vue globale, statistiques, gestion des utilisateurs et des validations.
- **Design Moderne** : Interface responsive, mode sombre/clair, animations fluides.
- **Visualisation de Données** : Graphiques interactifs pour le suivi de la croissance et de la répartition.

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 (Variables & Flexbox/Grid), JavaScript (Vanilla ES6+).
- **Backend / BDD** : Firebase (Firestore, Authentication).
- **Visualisation** : Chart.js.
- **Police** : 'Outfit' via Google Fonts.

## 📦 Installation et Lancement

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/plateforme-ibn-sina.git
   ```

2. Ouvrez le fichier `index.html` dans votre navigateur ou utilisez un serveur local (ex: Live Server sur VS Code).

## 🌍 Déploiement

Ce projet est prêt pour être déployé sur **GitHub Pages**.

1. Poussez le code sur la branche `main`.
2. Allez dans **Settings > Pages**.
3. Sélectionnez la source `main` (ou `master`) et sauvegardez.
